# datastruct_hw3

1. **作業系統**：x64 Ubuntu 15.10  
2. **IDE or 編輯器**：vim  
3. **Compiler**：g++ (Ubuntu 5.2.1-22ubuntu2) 5.2.1 20151010  
4. **執行步驟：**   
```
    輸入0~5選擇樣執行的動作：
        1. 再輸入一個值來插入資料到樹中。
        2. 輸入一個值，來刪除樹裡面的這比資料。
        3. 刪除樹裡面最大的值。
        4. 以Level order traversal來印出樹。
        5. 輸入一個節點的值，來算出這個節點下總共有幾個節點。
```
